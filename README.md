# Portfólio

Projeto do site com meu portfólio, ainda em fase inicial sendo feito em VUE.JS.

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
